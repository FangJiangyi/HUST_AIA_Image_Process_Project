# HUST_AIA_Image_Process_Projectd

本项目为华中科技大学人工智能与自动化学院自动化方向学生图像处理与模式识别课程设计，仅供华中科技大学学生内部参考。

## 项目概述

### 任务描述

- 实现不同形状工件的检测识别（硬币、螺栓、螺钉、螺母、L杆...）
- 实现工件的自动分类、计数、面积测量

### 任务特点与难点

* 图像采集任务
* 提取不变特征检测识别
* 解决工件粘连问题
* 保证实时性
